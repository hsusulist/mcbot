What Mod Do I Need?

To connect your Minecraft server console to Discord, you'll need a server-side mod that bridges Minecraft with Discord.

Recommended Mods:

1. DiscordSRV (Most Popular)
  • Platform: Spigot, Paper, Bukkit
  • Download: SpigotMC
  • Features:
    • Console output to Discord
    • Chat bridging between Minecraft and Discord
    • Player join/leave notifications
    • Death messages
2. Minecraft Discord Integration
  • Platform: Fabric, Forge
  • Download: CurseForge
  • Features:
    • Console logging
    • Player activity tracking
    • Server status updates
3. Discord Integration (Fabric)
  • Platform: Fabric
  • Download: Modrinth
  • Features:
    • Lightweight integration
    • Chat sync
    • Basic console output

Setup Instructions

  1. Download the mod for your server platform (Spigot/Paper/Fabric/Forge)
  2. Install it in your server's plugins/ or mods/ folder
  3. Restart your Minecraft server
  4. Configure the mod:
    • Add your Discord bot token in the mod's config file
    • Set the console channel ID (use Discord Developer Mode to get channel IDs)
    • Configure which events to log
  5. Restart the server again to apply changes
